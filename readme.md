# Invoice Processing RAG Application

This is a simple Retrieval-Augmented Generation (RAG) application designed to process invoices uploaded as PDF files. The application extracts data from the uploaded invoices and returns the extracted information in JSON format.

## Features

- Upload invoices in PDF format.
- Process invoices using RAG techniques.
- Extract and return invoice data in JSON format.

## Requirements

Ensure you have the required dependencies installed. Refer to the `requirements.txt` file for the list of dependencies.

## Usage

1. Place your invoice PDF files in the application directory.
2. Run the application script to process the invoices.
3. The extracted data will be returned in JSON format.

## File Structure

- `invoice-extraction.py`: Main script for processing invoices.
- `invoiceutil.py`: Utility functions for invoice processing.
- `requirements.txt`: List of dependencies.
- `readme.md`: Project documentation.
